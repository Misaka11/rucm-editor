## Prerequisites

### Java

You need Java 11 to use backend.

### Install nvm

```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.5/install.sh | bash
```

### Install npm and node

~~~bash
nvm install 10
nvm use 10
~~~

### Install yarn

~~~bash
npm install -g yarn
~~~
